## Logseq URL Title

Automatically fetches the title of a website and wraps it into markdown link format. Also, renders the favicon of the url next to it.

![demo](demo.gif)

To disable the favicon, override the `--favicons` variable.

```css
:root {
  --favicons: none;
}
```
